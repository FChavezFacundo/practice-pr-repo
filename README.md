# Practice PR Repo

This is a small practice repository used to learn the GitHub pull request workflow.

## What this repo does

It doesn't do much yet — it's a sandbox for practicing:

- Cloning and configuring a repo
- Creating branches
- Opening pull requests
- Reviewing and merging changes

## Getting started

1. Clone the repo: `git clone https://github.com/FChavezFacundo/practice-pr-repo.git`
2. Create a branch for your change: `git checkout -b my-change`
3. Commit your change and push: `git push -u origin my-change`
4. Open a pull request on GitHub
